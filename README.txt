D&D Инструменты для ведущего
D&D Инструменты — это набор удобных веб-приложений для настольных ролевых игр (например, Dungeons & Dragons), которые помогут ведущему и игрокам управлять предметами, монстрами, боями и картами прямо в браузере. Все данные сохраняются локально или экспортируются в формате JSON.

Состав набора:
1. Кузница предметов (Item Forge D&D)
Создание и редактирование собственных предметов с указанием редкости, типа, эффектов и других параметров.
Импорт и экспорт предметов в формате JSON.
Локальное сохранение списка предметов.
Markdown-подобное форматирование описаний.
Удобный и адаптивный интерфейс.
Файлы:
Item Forge 1.5.html — основной HTML
Script.js — логика
Styles.css — стили

2. Виварий (Vivarium D&D)
Создатель монстров и врагов для ваших сражений.
Полные статблоки, иммунитеты, сопротивления, особые свойства.
Загрузка и предпросмотр изображений существ.
Импорт/экспорт монстров в JSON.
Современный тёмный интерфейс.
Файлы:
Vivarium D&D.html
scripts.js
styles.css

3. Трекер боя (Battle Tracker D&D)
Быстрое добавление участников боя с инициативой, КД, ХП и др.
Применение урона/лечения, отслеживание концентрации и спасбросков смерти.
Подсветка активного участника, переход между ходами.
Импорт/экспорт состояния боя (JSON).
Автоматическое сохранение в браузере.
Файлы:
Battle_tracker 2.0.html
scripts.js
styles.css

4. Картограф (Map Builder D&D)
Создание и редактирование карт боевых полей (грид, стены, двери, объекты).
Интуитивные инструменты для рисования и удаления.
Сохранение и загрузка карт.
Удобный интерфейс под стилистику D&D.
Файлы:
Map Builder 1.0.html
scripts.js
styles.css

Как пользоваться
Откройте нужный HTML-файл в современном браузере (Chrome, Firefox, Edge и др.).
Вносите данные, создавайте предметы, монстров, карты или ведите бой.
Для сохранения или переноса данных используйте экспорт/импорт JSON.
Все изменения автоматически сохраняются в вашем браузере (localStorage).

Требования
Современный браузер (Chrome, Firefox, Edge и др.)
Интернет — только для загрузки шрифтов (опционально)

Импорт/Экспорт
Экспорт: сохраняет данные в файл формата JSON.
Импорт: загружает данные из файла JSON (структура соответствует экспортируемому файлу).

Автор - Merci_beaucoup (MerciBackdoor)
Инструменты созданы для личного использования ведущими и игроками D&D.

Примечание:
Все данные сохраняются только в вашем браузере. Для переноса на другое устройство используйте экспорт/импорт.

Удачных приключений и ярких сессий!
