# Кузница предметов D&D

Добро пожаловать в **Кузницу предметов D&D**! Этот проект позволяет создавать, редактировать и сохранять магические предметы для вашей кампании в Dungeons & Dragons.

## Функционал

- **Создание предметов**: Заполните форму, чтобы создать новый предмет.
- **Список предметов**: Все созданные предметы отображаются в боковой панели.
- **Редактирование предметов**: Нажмите на предмет в списке, чтобы отредактировать его.
- **Сохранение предметов**: Сохраните предметы в формате JSON на вашем устройстве.
- **Загрузка предметов**: Импортируйте предметы из JSON-файлов.
- **Локальное хранилище**: Все данные сохраняются в браузере и восстанавливаются при перезагрузке страницы.

## Установка

1. Скачайте или клонируйте репозиторий.
2. Откройте файл `Item Forge 1.5.html` в вашем браузере.

## Использование

1. **Создание предмета**:
   - Заполните поля формы, такие как название, редкость, тип и другие характеристики.
   - Нажмите кнопку **"Создать предмет"**.

2. **Редактирование предмета**:
   - Нажмите на предмет в списке, чтобы отобразить его детали.
   - Внесите изменения в форму и сохраните их.
   - Используйте *текст* для Italics и **текст** для Bold стилей.

3. **Сохранение предмета**:
   - Нажмите кнопку **"Сохранить"** в разделе отображения предмета.
   - Выберите место для сохранения JSON-файла.

4. **Загрузка предметов**:
   - Нажмите кнопку **"Загрузить"** и выберите один или несколько JSON-файлов с предметами.

5. **Очистка формы**:
   - Нажмите кнопку **"Очистить"**, чтобы сбросить все поля формы.

## Структура проекта

- **`Item Forge 1.5.html`**: Основной HTML-файл с интерфейсом приложения.
- **`Styles.css`**: Стили для оформления приложения.
- **`Script.js`**: Логика работы приложения.

## Требования

- Современный браузер с поддержкой JavaScript и LocalStorage.

## Советы

- Используйте кнопку **"Очистить"**, чтобы сбросить форму перед созданием нового предмета.
- Убедитесь, что загружаемые JSON-файлы соответствуют структуре предмета.
